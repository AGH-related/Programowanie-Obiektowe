# Programowanie Obiektowe

Repozytorium zawiera rozwiązania zadań realizowanych w ramach kursu **Programowanie Obiektowe**, prowadzonego na kierunku **Informatyka** na **AGH**.

## 📖 O projekcie

Kurs koncentruje się na praktycznym wprowadzeniu do paradygmatu programowania obiektowego. W ramach tworzonych projektów zaimplementowano i utrwalono mechanizmy charakterystyczne dla języka **Java**, takie jak:
* Klasy i obiekty
* Dziedziczenie
* Polimorfizm
* Interfejsy
* Obsługa wyjątków

Zadania zostały zrealizowane na podstawie instrukcji udostępnionych w repozytorium: [Soamid/obiektowe-lab](https://github.com/Soamid/obiektowe-lab).
